# pyDBO
This is an implementation in python of an object oriented model for a postgresql database

This piece of code, release under GPL v3 license, supports :
    - Creation of the object model
    - Description of the object properties
    - Instanciation of the object
    - Inheritance between the objects
    - Calculated properties, defined as SQL queries executed when accessed, with variables in SQL text.

Pieces planned to be added :
    - Exception management

The main.py file gives and example on how to use it and also serve as testing purpose.
